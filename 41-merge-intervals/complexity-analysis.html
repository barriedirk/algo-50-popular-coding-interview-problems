Complexity analysis

Solution 1:

We put n = length of intervals

Time complexity: O(nlogn)

Explanation:

We are sorting n intervals then traversing them twice by using for loops

T(n) = nlogn + 2n = O(nlogn)

Space complexity: O(n)

Explanation:

We are creating an output array that will contain n intervals of 2 elements

S(n) = 2n = O(n)
