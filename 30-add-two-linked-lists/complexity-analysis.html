Complexity analysis

Solution 1:

We put n = length of list1

             m = length of list2

Time complexity: O(max(n,m))

Explanation:

We keep looping while we didn't reach the end of list1 and the end of list2, so the number of iterations depends on the length of the longest linked list between list1 and list2, this is why the number of iterations is in O(max(n,m))

Space complexity: O(max(n,m))

Explanation:

We are creating a new node at each iteration, and we said earlier that the number of iterations is in O(max(n,m))
