Complexity analysis

Solution 1:

We put n = length of str

Time complexity: O(n^3)

Explanation:

The withoutRepeating() function has an O(n) cost where n is the length of the string passed as a parameter

In our solution function, the outer loop does n iterations, the inner loop does n-i iterations, and inside the inner loop, we are calling the withoutRepeating() function on substr, a string that has j-i characters, so the cost of an iteration of the inner loop is j-i

We can represent the cost with this sum:

Which is equal to 1/6(n^3 - n)

T(n) = 1/6(n^3 - n) = 1/6 n^3 - 1/6 n = O(n^3)

Space complexity: O(n)

Explanation:

Slicing a string creates a new one, and it's what we did in substr = str[i:j+1]

The greatest length of substr is n, when i is 0 and j is n-1, so we use n extra space:
S(n) = O(n)
