Complexity analysis

Solution 1:

We put n = number of nodes in the tree

             h = height of the tree

Time complexity: O(n²)

Explanation:

In the worst case, when the nodes form a linked list from the left, at each level, we will have to traverse all the remaining levels to take nodes to the right side

We have n levels because each node is in one level

So at the first level, the while loop will do n-1 iterations, then n-2 iterations in the second level, and so on

n+(n-1)+(n-2)+...+0 = n(n-1)/2 = (n²-n)/2 = (1/2n²)-(1/2n) = O(n²)

Space complexity: O(h)

Explanation:

The maximum call stack size depends on the height of the tree
