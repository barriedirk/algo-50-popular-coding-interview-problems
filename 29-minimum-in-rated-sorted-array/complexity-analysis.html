Solution 3:

We put n = length of arr

Time complexity: O(logn)

Explanation:

We are using the decrease and conquer method same as in binary search, we are dividing the input size by 2 at each call, we get an O(logn) time complexity

Space complexity: O(1)

Explanation:

Our recursive function is tail recursive, it doesn't need to go back to previous calls, so we don't need extra space to save the calls
