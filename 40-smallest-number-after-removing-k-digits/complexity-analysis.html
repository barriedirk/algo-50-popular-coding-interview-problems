Complexity analysis

Solution 1:

We put n = length of num

              k = number of digits to remove

Time complexity: O(n)

Explanation:

Even we have nested loops, the inner while loop will do at most n iterations during the whole process, because each digit is popped only once, and we have n digits.

So all the loops will do at most n iterations, we get a time complexity of O(n)

Space complexity: O(n)

Explanation:

We are using a stack that can contain n elements
