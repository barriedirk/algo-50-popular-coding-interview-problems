Complexity analysis

Solution 1:

We put n = number of nodes in the tree

             h = height of the tree

Time complexity: O(h)

Explanation:

Each recursive call is calling the function only once, and it's moving to the next level, so the number of recursive calls depends on the height of the tree, and each call has an O(1) cost, this is why we get an O(h) time complexity

Space complexity: O(h)

Explanation:

The maximum call stack size is h+1, this is why we get an O(h) space complexity
