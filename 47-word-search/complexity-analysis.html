Complexity analysis

Solution 1:

We put n = number of rows of the grid

             m = number of columns of the grid

             w = length of word

Time complexity: O(n*m*(4^w))

Explanation:

For the recursive function, we will use the recurrence relation method

In the base case, when counter is equal to w, we just return a boolean, so

T(n,m,0) = 1

In the recursive case, we recursively call the function 4 times starting from the next character (once for each direction), and remaining stuff is O(1), so:

T(n) = 4T(n,m,w-1)+1

We keep replacing:

T(n,m,w) = 4(4T(n,m,w-2)+1)+1

T(n,m,w) = 16T(n,m,w-2) + 5

T(n,m,w) = 16(4T(n,m,w-3)+1)+5

T(n,m,w) = 64T(n,m,w-3) + 21

T(n,m,w) = 64(4T(n,m,w-4)+1)+21

T(n,m,w) = 256T(n,m,w-4) + 85

T(n,m,w) = 256(4T(n,m,w-5)+1)+85

T(n,m,w) = 1024T(n,m,w-5) + 341
.
.

T(n,m,w) = (4^k)T(n,m,w-k) + (4^k - 1)/3
.
.

T(n,m,w) = (4^w)T(n,m,w-w) + (4^w - 1)/3

T(n,m,w) = (4^w)T(n,m,0) + (4^w - 1)/3

T(n,m,w) = 4^w + (4^w - 1)/3

T(n,m,w) = 4/3(4^w) - 1/3

T(n,m,w) = O(4^w)

And in the main solution function, we are calling this function for each one of the n*m cells of the grid

T(n,m,w) = n*m*O(4^w) = O(n*m*(4^w))

Space complexity: O(w)

Explanation:

The maximum call stack size is w+1 and the visited set contains up to w tuples of 2 elements (i and j)

S(n,m,w) = w+1+2w = 3w+1 = O(w)
