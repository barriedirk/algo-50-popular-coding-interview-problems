Complexity analysis

We put n = the number of nodes in the tree

Time complexity: O(n)

Explanation:

Except calling the function again, a recursive call costs O(1), because we just print an integer or return, and we will have n recursive calls, one for each node, so:
T(n) = n*O(1) = O(n)

Space complexity: O(h)

Explanation:

A recursive call isn't using input-size related extra space, but the call stack can contain up to h+1 stack frames, where h is the height of the tree, because if the height of a tree is 3 for example, then we have 4 levels, the call stack will contain a stack frame from the level 0, a stack frame from the level 1, a stack frame from the level 2, and a stack frame from the level 3, so:

S(n) = h+1 = O(h)
