Complexity analysis

We put n = length of str

Time complexity: O(n)

Explanation:

The first for loop is traversing the characters of str while incrementing, the cost is n

The second for loop is traversing the occurrences array, which has 128 elements, the cost is 128

T(n) = n + 128 = O(n)

Space complexity: O(1)

Explanation:

We are using the occurrences array, which has 128 elements, and some variables, but the extra space size is still constant, so:

S(n) = O(1)
