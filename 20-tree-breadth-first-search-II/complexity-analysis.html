Complexity analysis

We put n = the number of nodes in the tree

Time complexity: O(n)

Explanation:

Each node enters the queue only once, so the queue can have at most n elements, which means that the number of iterations of the while loop is n. And each iteration has a constant cost, so:

T(n) = n*O(1) = O(n)

Space complexity: O(n)

Explanation:

The queue can contain at most n nodes, and we are creating an array that will contain n nodes at the end, the extra space size is 2n, so:

S(n) = n+n = 2n = O(n)
