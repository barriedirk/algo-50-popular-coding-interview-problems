Complexity analysis

Solution 1:

We out n = length of str

Time complexity: O(n)

Explanation:

We are traversing the n characters if str, and at each iteration, we are doing O(1) operations (checking if bracket is in openingBrackets costs 3 because openingBrackets has only 3 elements, which is a constant)

T(n) = n*O(1) = O(n)

Space complexity: O(1)

Explanation:

We are using a stack that can contain n elements

S(n) = n = O(n)
