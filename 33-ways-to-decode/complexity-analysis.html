Complexity analysis

Solution 1:

We put n = length of str

Time complexity: O(f^n)

Explanation:

We have T(0) = 1 and in the worst case T(n) = T(n-1)+T(n-2), which is the Fibonacci function, whose the tight upper bound is in O(f^n), where f is the golden ratio (1.618033...)

Space complexity: O(n)

Explanation:

The maximum call stack size is n+1 because in the worst case, we jump by 1 character in the string only at each level (0, 1, ..., n-1, n)

S(n) = n+1 = O(n)


Solution 2:

We put n = length of str

Time complexity: O(n)

Explanation:

We are just traversing the characters of str while doing O(1) operations

T(n) = n*O(1) = O(n)

Space complexity: O(1)

Explanation:

We are not using input-size related extra space
