Complexity analysis

Solution 1:

We put n = length of arr

Time complexity: O(n^3)

Explanation:

The first for loop is repeated n times, the second for loop (the one inside the first one) is repeated n-i times, the third for loop (the one inside the second one) is repeated j-i+1 times, and what's inside the third for loop costs O(1)
We can represent the cost with this sum:

Which is equal to 1/6(2n + 3n² + n^3)

T(n) = 1/6(2n + 3n² + n^3) = 1/3 n + 1/2 n² + 1/6 n^3 = O(n^3)

Explanation:

We are using some integer variables only: maxSum, i, j, k, and actualSum, so:

S(n) = 5 = O(1)


Solution 2:

We put n = length of arr

Time complexity: O(n²)

Explanation:

The outer loop is repeated n times, the inner loop n-i times, and what's inside the inner loop costs O(1)

Cost of iterations of the outer loop:

1st iteration -> i = 0 -> n-i = n

2nd iteration -> i = 1 -> n-i = n-1

3rd iteration -> i = 2 -> n-i = n-2

.

.

nth iteration -> i = n-1 -> n-i = 1

So T(n) = n+(n-1)+(n-2)+...+1, which is the sum of integers from n to 1, which is n(n+1)/2

T(n) = n(n+1)/2 = (n²+n)/2 = n²/2 + n/2, we take the greatest term, we remove the constant, and we get a time complexity of O(n²)

Space complexity: O(1)

Explanation:

We are using some integer variables only: maxSum, i, j, and cumulativeSum, so:

S(n) = 4 = O(1)
