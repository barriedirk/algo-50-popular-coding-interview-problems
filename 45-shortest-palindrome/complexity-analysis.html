Complexity analysis

Solution 1:

We put n = length of str

Time complexity: O(n²)

Explanation:

Getting the substring from 0 to i costs i, so at the first iteration it costs 0, then 1 in the second iteration, and so on until n-1

0+1+...+(n-2)+(n-1) = n(n-1)/2 = (n²-n)/2 = (1/2n²)-(1/2n) = O(n²)

Space complexity: O(1)

Explanation:

We are not using input-size related variables
