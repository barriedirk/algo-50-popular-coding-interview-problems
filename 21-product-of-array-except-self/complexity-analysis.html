Complexity analysis

Solution 1:

We put n = length of arr

Time complexity: O(n²)

Explanation:

We are using nested for loops where the outer one goes from 0 to n (it does n iterations) and same thing for the inner one. And inside them we have O(1) operations, so:

T(n) = n * n * O(1) = O(n²)

Space complexity: O(n)

Explanation:

We are creating an array output that will contain n elements at the end

S(n) = n = O(n)


Solution 2:

We put n = length of arr

Time complexity: O(n)

Explanation:

cost of creating cumulativeFromLeft array: n

cost of the first for loop (it goes from 1 to n): n-1

cost of creating cumulativeFromRight array: n

cost of the second for loop (it goes from n-2 to -1): n-1

cost of creating output array: n

cost of the third for loop (it goes from 0 to n): n

total: T(n) = n+(n-1)+n+(n-1)+n+n = 6n-2 = O(n)

Space complexity: O(n)

Explanation:

We have 3 arrays of n elements: cumulativeFromLeft, cumulativeFromRight, and output, so:

S(n) = 3n = O(n)
